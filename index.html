Guide

You cannot play sudoku directly yet. You can adjust the screen size by dragging the top border of this little display box. This increases the height and in turn increases the size of each tile. Opening up the console will show you the solution using numbers. The numbering goes from 0-9, in order that they appear on the official HoloID page.


You can customize the number of holes in the board.
